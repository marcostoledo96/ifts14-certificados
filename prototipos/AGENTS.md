# AGENTS.md — Prototipos

## Propósito

Esta carpeta guarda capturas, prompts y experimentos visuales.

## Reglas

- No tratar prototipos como implementación final.
- Guardar origen de cada diseño: Stitch, v0 u otra herramienta.
- Si una pantalla se aprueba visualmente, documentar la decisión en `docs/frontend`.
- No mezclar archivos temporales con código productivo.
