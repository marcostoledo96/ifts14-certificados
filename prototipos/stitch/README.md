# Prototipos Google Stitch

Guardar acá capturas, exports o referencias generadas en Google Stitch.

## Reglas

- Nombrar archivos por pantalla.
- Guardar variantes elegidas y descartadas si aportan contexto.
- Documentar brevemente por qué se eligió una dirección.

Ejemplo:

```txt
validacion-publica-valida-v1.png
admin-dashboard-v2-elegida.png
pdf-horizontal-qr-v1.png
```
