# Sistema visual y anti-clichés

## Público

Folio técnico institucional / acta académica verificable.

## Admin

Mesa de trabajo de Bedelía / archivo institucional digital.

## Paleta

- Azul noche: `#0B1F33`
- Azul técnico: `#1565C0`
- Cian circuito: `#00A8C6`
- Verde validación: `#2E7D32`
- Gris papel: `#F5F7FA`
- Gris texto: `#263238`
- Blanco: `#FFFFFF`
- Ámbar: `#F9A825`
- Rojo revocado: `#C62828`

## Evitar

- glassmorphism;
- blobs;
- gradientes genéricos;
- cards blancas repetidas;
- hero SaaS;
- badges genéricos;
- íconos decorativos en cada bloque;
- estética fintech;
- dashboard comercial.

## Usar

- folios;
- sellos;
- bandas de estado;
- líneas finas;
- tablas/listas sobrias;
- zona de trazabilidad QR;
- textura técnica mínima.
