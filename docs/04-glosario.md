# Glosario

- **Alumno**: persona que recibe la certificación.
- **Curso**: actividad formativa que tiene fechas.
- **Fecha del curso**: día específico del curso.
- **Asistencia presente**: marca de que el alumno participó en una fecha.
- **Certificación**: constancia de alumno + curso + fechas presentes.
- **Token**: código seguro usado en el link/QR público.
- **QR**: código que abre `/validar/:tokenCertificacion`.
- **PDF complementario**: certificado horizontal con QR y datos institucionales.
- **Reenvío**: nuevo envío del PDF/aviso con el mismo QR.
- **Revocación**: cambio que hace que la certificación ya no se muestre como válida.
