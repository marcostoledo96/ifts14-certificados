# Guía para AGENTS.md

## Objetivo

Mantener instrucciones mínimas, claras y no contradictorias.

## Buenas prácticas

- Dar reglas de lectura.
- Evitar repetir todo el proyecto en cada AGENTS.
- Definir límites de la carpeta.
- Mantener idioma y estilo.
- No incluir listas gigantes de reglas que no aplican.

## Evitar

- Context bloat.
- Instrucciones contradictorias.
- Reglas técnicas no confirmadas.
- Copiar el mismo bloque en todas las carpetas.
