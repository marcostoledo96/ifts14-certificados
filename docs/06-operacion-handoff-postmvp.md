# Operación, handoff y post-MVP

## Guía rápida para Bedelía

1. Iniciar sesión.
2. Revisar panel.
3. Crear o editar curso.
4. Agregar fechas.
5. Registrar asistencias presentes.
6. Emitir certificación.
7. Descargar o enviar PDF.
8. Reenviar si hubo cambios.
9. Revocar si corresponde.

## Soporte básico

Problemas frecuentes:

- link no abre;
- QR no escanea;
- email no llega;
- datos incorrectos;
- PDF desactualizado;
- token inexistente;
- certificación revocada aparece como válida.

Cada problema debe registrarse con pasos para reproducir y evidencia.

## Handoff técnico

El futuro mantenedor debe recibir:

- acceso al repo;
- variables de entorno;
- instrucciones de deploy;
- datos demo;
- documentación de contratos;
- guía de seguridad;
- pendientes post-MVP.
