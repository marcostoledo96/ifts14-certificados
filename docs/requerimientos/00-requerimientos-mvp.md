# Requerimientos MVP

Ver `docs/02-requerimientos-reglas-negocio.md` para reglas principales.

## Funcionalidad mínima

- Validación pública.
- Login admin.
- Cursos y fechas.
- Alumnos.
- Asistencias presentes.
- Emisión directa.
- PDF horizontal con QR.
- Envío/reenvío.
- Revocación.
- Datos demo.

## No comprometido

- Carga masiva real.
- Integración web oficial.
- Google Login.
- Perfil público del alumno.
